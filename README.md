# msPatch
This utility allows you to pull down MS Patch Tuesday patches.

It applies the CTI filter on it for RCE and Exploitability of more likely or exploited.

# Requirements

1. requests
2. json
3. pandas
4. datetime
5. time
6. os
7. argparse
8. configparser

OPTIONS:
"-i" allows for the program to be run interactively.
